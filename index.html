txt2sound.js

txt2sound.js converts text to live audio:

txt2sound.setText('this is my text');
txt2sound.play();
txt2sound.stop();

txt2sound.js supports a large variety of synthesis and effect controls:

// set tempo
txt2sound.setTempo(120);

// set wave shape
txt2sound.setWaveShape('triangle');
txt2sound.setWaveShape('square');
txt2sound.setWaveShape('sine');

// set reverb
txt2sound.setReverbWet(0.5);
txt2sound.setReverbDry(0.8);
txt2sound.setReverbSize(0.23);

// set LFO
txt2sound.setLfoFrequency(14.5);
txt2sound.setLfoAmount(0.3);
txt2sound.setLfoWaveShape('sawtooth');

// set envelope
txt2sound.setAttack(10);
txt2sound.setRelease(100);
txt2sound.setNoteLength(200);

dependencies

txt2sound.js depends on audiolib.js. You can include audiolib.js from the external folder in this repository or obtain it directly from https://github.com/jussi-kalliokoski/audiolib.js.
